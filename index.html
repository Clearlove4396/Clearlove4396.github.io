hexo+github: creat a blog

Posted on 2020-10-23

Github Pages

  1. 创建一个仓库，仓库名：{your github name}.github.io
  2. 勾选Initialize this repository with a README

hexo

hexo是一个博客框架，可以把本地.md文件生成网页。Github Pages其实是一个挂载容器，hexo可以把生成的网页挂载到github上显示。

操作步骤

  1. 安装node.js

  2. 安装Git

  3. 安装Hexo

    • npm install -g hexo-cli

  4. 创建本地博客

    • 创建文件夹blog（需要是空文件夹）
    • 鼠标右键blog，选择Git Bush Here
    • hexo init初始化
    • 输入npm install安装相关依赖包
    • hexo g（即generate）把.md文件转化为.html文件
    • hexo s创建服务，此时浏览器中输入localhost:4000可以看到网页

  5. 文件夹简单说明

    • _config.yml博客的配置信息
    • source/_posts存放博客的原文件，也就是.md文件及一些别的东西，比如图片资源
    • themes博客的界面主题
    • public生成的.html文件，将来也是把这个东西上传到Github上部署

  6. 把本地博客部署到Github上

    • 打开_config.yml，找到deploy: 并进行相关修改，注意这里的branch写成main，因为github上面的仓库的主分支叫做main，而不是原来的master
    • 需要执行一次（只需要一次）：npm install hexo-deployer-git --save
    • 执行hexo d，得到INFO Deploy deno: git即为部署成功

  7. 更换博客主题

    • 去https://hexo.io/themes/上面找一个主题，然后git clone到themes问价夹下，假如文件夹的名字叫jiangnansytle
    • 在_config.yml中把theme改成jiangnansytle即可
    • 如果更换主题后发现生成的html为空，则可能是主题有问题，换一个把。

  8. 在.md中添加图片

    • 参考下面两个链接就好了

      https://www.jianshu.com/p/f72aaad7b852

      https://www.jianshu.com/p/3db6a61d3782
